## Purpose
	Authentification server with creation of a JSON Webtoken (jwt) transfered here by cookies.

## Requirements
	-	Node.js
	-	express
	-	body-parser
	-	jsonwebtoken
	-	express-jwt
	
## Use
	Open a terminal in "src",
        Execute "npm install"
        Execute "npm run-script build"
	Do "node serveur.js",
	Open "localhost:8080" in any web browser
